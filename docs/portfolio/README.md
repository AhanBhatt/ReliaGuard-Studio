# Portfolio Positioning

## LinkedIn project description

ReliaGuard Studio is a production-style AI evaluation platform for detecting unsafe human-AI decision behavior. It analyzes when AI assistance improves average accuracy while still producing harmful overreliance on wrong advice or underreliance on correct advice. The platform integrates five public datasets spanning 43,263 records from 2,229 participants/students, exposes FastAPI endpoints for reliance-state prediction and explanation, and includes a Next.js dashboard for case simulation, conformal gating, model evaluation and policy simulation.

## Resume bullets

- Built ReliaGuard Studio, a full-stack AI evaluation platform using FastAPI, Next.js, TypeScript and Python ML pipelines to detect overreliance, underreliance and unsafe human-AI decision behavior.
- Integrated five public human-AI datasets covering 43,263 records and 2,229 participants/students; implemented canonical schemas, reliance-state labels, GEE statistics, calibration analysis, cross-dataset transfer, conformal risk control and sensitivity audits.
- Designed interpretable ReliaGuard-NS API endpoints for calibrated risk, rule traces, counterfactual explanations, conformal thresholds, ablations, policy simulation and model-card reporting.
- Added production engineering scaffolding including Docker Compose, GitHub Actions, typed frontend pages, API schemas, model cards, reproducibility docs and release safety boundaries.

## Portfolio tagline

AI systems can improve average accuracy while hiding harmful reliance failures. ReliaGuard Studio makes those failures measurable, explainable and ready for prospective validation.
