# ReliaGuard Studio Web

Next.js + TypeScript + Tailwind frontend for the ReliaGuard Studio product surface.

Pages:

- `/` landing page.
- `/simulator` interactive reliance-state case simulator.
- `/gating` conformal threshold dashboard.
- `/evaluation` model-evaluation lab.
- `/policy` policy simulation.
- `/data` data and reproducibility page.
- `/limitations` safety and limitations.

Run after installing Node dependencies:

```powershell
cd apps/web
npm install
npm run dev
```
